-Run program by typing make -f makfile. 

-Make sure that there is an internet connection as the program makes http requests to retrieve the sgm files.

-Once processing is complete (takes a little while) all chosen attributes are displayed on the screen along with the number of times they appear in the entire set of articles (global count)

-The actual feature vectors are written to a file called featureVector in the same directory. The class labels are topics;places, document id

-The simple feature vectors are written to a file called featureVectorSimple. the class labels remain the same
